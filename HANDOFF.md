# MELT VERSE ― サイト制作 引き継ぎメモ

このファイルは、チャットで決めたサイトの設計方針をまとめたものです。
Claude Code など別の場所で続きを作るときは、このファイルを一緒に読ませてください。

---

## サイトの目的
- 管理人 **melt** が、AIチャットボットアプリ **zeta** に投稿しているプロットを紹介し、
  それを原案としたオリジナル小説・キャラクター・世界観の設定をまとめる展示サイト。
- これまで piku.page（`piku.page/@meltverse_`）で運用。今回、自前の HTML/CSS で作り直す。

## 全体のデザイン
- 配色：背景 `#232323` / 文字 `#FFFFFF`（ダーク基調）。
- レイアウト：中央 1 カラム（最大 640px）。スマホ対応。
- フォント（暫定・差し替え可）：見出し = Archivo / 日本語 = Zen Kaku Gothic New。
- 主要な色・フォントは `index.html` 冒頭の CSS 変数（`:root`）で一括管理。

## トップページ（index.html｜完成済み）
縦の並び：
1. アイキャッチ画像（**445×445 の正方形**・角丸）
2. サイトタイトル「MELT VERSE」
3. 紹介文：「zetaで公開中のプロットの紹介・キャラクターと世界観の設定・オリジナル小説をまとめたサイトです。」
4. SNS アイコン（**X** ＋ **zeta**。zeta はアプリのアイコン画像を `images/` に置いて読み込み）
5. 「- Plot List -」見出し（中央寄せ）
6. プロット一覧（下記カード）
7. フッター

### Plot List のカード（案A・横並び）
1 プロット = 1 枚のカード。中身は次の 3 要素：
- **バナー**：zeta の該当プロットへの外部リンク。下に「zetaでトークする ↗」を添える。
- **短い紹介文**：1〜2 行。
- **「詳細を見る →」**：サイト内の詳細ページへのリンク。

※ リンクはこの 2 つ（バナー＝zeta外部／詳細＝サイト内）だけに集約。
※ `index.html` 内の `<article class="plot-card">…</article>` をコピーして増やす。

### フッター
- piku 標準の「LOGO」「Footer Link」は**削除済み**。
- テンプレート製作者表記「**Template Design /@Yaz**」は**残す**。
- **管理人名 melt** ＋ **お問い合わせ**リンク（メールフォームは未定。今は `href="#"` のまま）。

## 詳細ページ（これから作る）
- **1 ページ完結**。ページ内を **[キャラクター] [世界観] [小説]** の 3 タブで切り替える（設定が主役）。
- 冒頭に大きめバナー（zetaリンク再掲）＋タイトル＋あらすじ。

### キャラクタータブ
- 1 キャラ = 1 カード。上に**横スクロールの挿絵ギャラリー**（プラグイン使用予定）、下に**名前＋説明文**。
- プロフィールは**すべてテキストにまとめる**（年齢などのタグや「CODENAME ○○」ラベルは使わない）。

### 世界観タブ
- トピックごとにカード（挿絵＋見出し＋本文。挿絵なしの本文だけのカードも可）。
- 末尾に**用語集**を定義リスト（用語｜説明）で掲載。

### 小説タブ
- 1 プロットに小説は **0〜複数**。
- 各話を**リスト表示**（話数・タイトル／一行あらすじ／投稿日・文字数などのメタ）。
- 小説が 0 本のときは**空表示**（「小説はまだありません」等）を出す。

## 画像について
- 画像は管理人が自分で用意して差し込む。
- `index.html` と同じ階層に `images/` フォルダを作り、`アイキャッチ画像` / `banner` の
  プレースホルダーを `<img>` に差し替える（各所にコメントで例を記載済み）。
- zeta アイコンは `images/zeta.png` を読み込む設定。
